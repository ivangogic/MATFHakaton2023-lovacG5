## Requirements

### Running the **desktop** application
Dependencies: Python, PyQT5, Pygame
```
pip install pygame
pip install PyQt5
```

### Running the **web** app
Dependencies: FastAPI
```
pip install fastapi
pip install "uvicorn[standard]"
```
